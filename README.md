# Expo Router starter project

A plain project with the following alredy set up:

- Expo Router
- Nativewind
- ESLint & Prettier
- TypeScript

And one utility:

- `src/utils/cn.ts` for merging Tailwind classes

## Start a new project with this template

```sh
# with npm
npx create-expo-app appName --template https://github.com/DanielIurasoc/expo-router-starter
```
